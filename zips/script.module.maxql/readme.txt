MaxQL
Description:
With 1-click this add-on will convert the max quality filter for all supported add-ons to either 1080P or 4K.


Instructions for adding this repo in Kodi:
Open the Kodi File Manager
Select "Add source"
The path for the source is https://zaxxon709.github.io/repo/ (Give it the name "709REPO")
Conversion Built-in Commands:
Convert to 1080P
RunPlugin(plugin://script.module.maxql/?mode=1)

Convert to 4K
RunPlugin(plugin://script.module.maxql/?mode=2)
